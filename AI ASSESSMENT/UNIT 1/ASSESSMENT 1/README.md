Objective
To implement the Uniform Cost Search (UCS) algorithm for finding the least-cost path between the starting warehouse and the destination warehouse in a logistics delivery network.
Software Requirements
•	Python 3.x 
•	Python IDLE / VS Code / PyCharm 
Files Included
•	Problem.pdf – Uniform Cost Search problem statement. 
•	Solution.pdf – Detailed explanation of the UCS algorithm and search process. 
•	Python_Code.py – Python implementation of the UCS algorithm. 
•	Output.png – Screenshot of the program output. 
•	Report.pdf – Complete assessment report. 
•	README.md – Project documentation. 
How to Run
1.	Open Python_Code.py in Python IDLE or any Python IDE. 
2.	Run the program. 
3.	The program explores the weighted graph using Uniform Cost Search. 
4.	It displays the least-cost path from S to G along with the total travel cost. 
Expected Result
The program successfully applies the Uniform Cost Search (UCS) algorithm to identify the optimal path from the source warehouse S to the destination warehouse G. By always expanding the node with the lowest cumulative path cost, UCS guarantees the least-cost solution for the given delivery network.
